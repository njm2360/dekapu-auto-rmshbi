# でかプ お部屋シャルベ回収自動化ツール (OpenCV版)

## 動作環境

あらかじめ[uv](https://github.com/astral-sh/uv)を導入してPythonをインストールしてください
動作確認済みのPythonバージョンは3.13.9です

モニターの解像度は1920x1080(フルHD)以上を推奨します

VRChatのグラフィック設定からFOVを必ず60度に設定してください

使用するアバターは透明アバターで身長を1m程度に設定してください

## 使用方法

あらかじめVRChat側はクイックメニューを閉じた状態で以下のコマンドを実行してください
(ESCキーが緊急停止ボタンを兼ねているため開いたままだと閉じるときに停止します)
※ウィンドウ間の移動にはALT+TABを使用してください

`uv run main.py`で実行できます

VRChatのウィンドウに移動したらF5キーを押すとウィンドウの確定がされ、ウィンドウサイズが強制的に1024x768にリサイズされます

2階のタスクパネルの対角の角に立ったらカーソルをタスク獲得ボタンに合わせてF6を押すと開始です。動作中はマウスカーソルを動かさないでください

F6を再度押すとオンオフの切り替え、ESCキーを押すと緊急停止です(通常終了時もESCキーです)
